What is DragonKnowledge â€‹

Dragon Knowledge Framework is an innovative framework for the PHP programming language. It stands out for its ease of use and intuitive operation, providing a solid foundation for website creation.

Why use it? â€‹

Simplicity of use â€‹

The framework offers a clear, accessible structure, enabling developers to get their web projects off the ground quickly and efficiently. The index page visible to users is moved to the /public/ directory, reinforcing security by preventing access to the site's backend and any sensitive files.

Integrated security â€‹

Dragon Knowledge Framework incorporates robust security measures. By moving the index page to the /public/ folder, it protects the backend and critical files, reducing the risk of compromise.

Enhanced productivity â€‹

To maximize productivity, the framework offers SEO preconfiguration, optimizing your site's natural referencing right from the start. What's more, essential PHP elements are pre-installed, including PHP, SCSS and other files, facilitating development.

Support for modern tools â€‹

The framework is compatible with NodeJS and Composer, although their use is optional. This compatibility gives developers the flexibility to integrate modern tools according to their specific needs, without constraint.

Language support â€‹

DragonKnowledge supports all the languages you use. But it also comes preconfigured for SCSS.

Dragon Knowledge Framework is the ideal choice for PHP developers who want to combine simplicity, security and productivity in their web projects.
